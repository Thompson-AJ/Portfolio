LCC First Responder Map

This map was designed as part of the Pre Plan Project. While it does not follow traditional cartographic principles, It is a highly functional map that provides the exact necessary details that first responders need when arriving at this location. It was designed to be extremely high contrast, as simple as possible while combining as much information as is needed, and had to be as to scale as possible. Once challenge was designing a symbology for the roads that didn't require a key, but also allows first responders to know what the accesibilty is in the area.
