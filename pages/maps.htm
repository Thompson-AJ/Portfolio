This project actually started out as a challange to a cautionary tale. The GE for one of my cartography classes told us that no one should really ever use hot pink in their maps.

LCC Pre Plan Map

This map was created as part of the Pre Plan Project as a quick reference map for firefighters, it was designed to be as practical and easy to read as possible from a moving fire truck by first respoonder, not a typical cartographic design.
